# AC-movies

## 環境需求

- **作業系統**: Windows
- **Node.js 版本**: v20.17.0
- **套件管理工具**: npm
- **依賴套件**:
  - express
  - express-handlebars

## 安裝與執行步驟 

1. **Clone 專案至本地端**  
   使用以下命令將專案Clone到本地端：
   
   git clone https://github.com/your-username/AC-movies.git

2. **安裝需要套件**
   
   git install
   
3. **啟動開發伺服器**
   
   npm run dev
4. **啟動正式伺服器**
   
   npm start
## 功能描述

此專案提供以下功能：

- **瀏覽電影清單**
- **搜尋電影**
- **查看電影資訊**
- **電影上映日期**
- **電影簡介**
